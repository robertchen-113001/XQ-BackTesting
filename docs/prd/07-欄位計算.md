## Spec 格式問題標註

> ⚠️ **下標符號 OCR 殘影**：原始 Spec 多處以逗號（,）替代下標 `t` 或 `t-1`，例如：
> - `最大投入RN%,=（淨利,-淨利,,）/當日最大投入金額,` → 應為：`最大投入RN%ₜ = (淨利ₜ − 淨利ₜ₋₁) / 當日最大投入金額ₜ`
> - `時間加權報酬率TWR%, = (1 + TWR%, 1) * (1 + 時間加權RN%, ) - 1` → 應為：`TWRₜ% = (1 + TWRₜ₋₁%) × (1 + 時間加權RN%ₜ) − 1`
> 本 PRD 已依語意修正標記方式，所有下標以 `ₜ`（當日）和 `ₜ₋₁`（前一日）表示。

> ⚠️ **缺少運算符號**：多處公式在計算項目之間缺少「−（減號）」，例如：
> - 「當日淨利ₜ = 淨利ₜ 淨利ₜ₋₁」→ 應為「當日淨利ₜ = 淨利ₜ **−** 淨利ₜ₋₁」
> - 「拉回ₜ = IFF(淨利ₜ < 淨利高點ₜ, 淨利ₜ 淨利高點ₜ, 0)」→ 應為「**淨利ₜ − 淨利高點ₜ**」
> 本 PRD 已補入推斷的運算符號，**標示為【補】**，建議原始文件作者確認。

> ⚠️ **疑似邏輯錯誤**：最大區間虧損公式中，`報酬率高點ₜ = MIN(報酬率ₜ, 報酬率高點ₜ₋₁)` 使用 MIN，但依語意應為 MAX（記錄歷史報酬率最高點作為回撤基準）。此處標記為【需確認】，建議與計算工程師核對。

## 1. 計算參數

### 1.1 初始資金

- 用於處理最大同時進場交易筆數的資金限制

### 1.2 槓桿倍數

| 設定類型 | 行為 |
|---------|------|
| 固定數量（等量）| 開倉金額縮小，交易數量不變 |
| 固定金額（等額）| 開倉金額不變，交易數量放大 |

| 商品類型 | 槓桿計算 |
|---------|---------|
| 股票 | 槓桿倍數 = 100%（無槓桿）；若啟用融資：`1 / (1 − 融資比例)`，例：`1 / (1 − 60%) = 2.5 倍` |
| 期貨 | 槓桿倍數 = `1 / 保證金成數`，例：`1 / 13.5% ≈ 7.41 倍` |

## 2. 基礎計算量

### 2.1 交易數量 Qty

- 帶多空符號，**保留 6 位小數，超過部分捨去**

| 類型 | 說明 | 預設值 |
|------|------|--------|
| 原始 | 自動交易中使用者自訂 | — |
| 等量 | 固定張數 | 1 張（美股 100 股）|
| 等額 | 固定投入金額 | 100 萬（美股 1 萬美元）|
| 等比 | **不支援**交易數量 | — |

**槓桿交易數量** = `交易數量 × IFF(固定數量, 1, 槓桿倍數)`

### 2.2 Lots

系統根據商品定義自動帶入。

### 2.3 開倉金額

```
開倉金額 = ROUNDDOWN(進場價格 × ABS(交易數量) × Lots / IFF(固定數量, 槓桿倍數, 1), 0)
```

- 以進場價計算的交易金額
- 不帶多空方向（永遠為正值）
- **無條件捨去**至整數位

### 2.4 價差

```
價差 = ROUNDDOWN(
  (出場還原價closePrice2 × 進場還原率 − 進場原始價costPrice)
  × IFF(作多, 1, −1) × ABS(交易數量) × Lots,
  0
)
```

**進場還原率**：
- 個股：`進場原始價costPrice / 進場還原價costPrice2`
- 期貨：期貨還原因子

**累計股息**：
- 個股：區間內除息總額
- 期貨：區間內期貨還原點數總額

> 📌 **待確認**：「價差（股息不再投入）」的計算方式仍待確認，公式暫標為 Draft。

### 2.5 交易成本

**股票交易成本** = 買進成本 + 賣出成本

```
買進成本 = ROUNDDOWN(
  IFF(作多, 進場原始價, 出場還原價 × 進場還原率) × 手續費% × ABS(交易數量) × Lots,
  0
)

賣出成本 = ROUNDDOWN(
  IFF(作多, 出場還原價 × 進場還原率, 進場原始價) × 手續費% × ABS(交易數量) × Lots,
  0
)
+ ROUNDDOWN(
  IFF(作多, 出場還原價 × 進場還原率, 進場原始價) × 交易稅% × ABS(交易數量) × Lots,
  0
)
```

**期貨交易成本** = 買進成本 + 賣出成本

```
買進成本 = ROUNDDOWN(手續費$ × ABS(交易數量), 0)
         + ROUNDDOWN(IFF(作多, 進場原始價, 出場還原價 × 進場還原率) × 期貨交易稅% × ABS(交易數量) × Lots, 0)

賣出成本 = ROUNDDOWN(手續費$ × ABS(交易數量), 0)
         + ROUNDDOWN(IFF(作多, 出場還原價 × 進場還原率, 進場原始價) × 期貨交易稅% × ABS(交易數量) × Lots, 0)
```

### 2.6 獲利金額

```
獲利金額 = 價差 − 交易成本
```

## 3. 報酬率計算

### 3.1 當日報酬率 RNₜ

**最大投入 RNₜ%**：

```
最大投入RN%ₜ = (淨利ₜ − 淨利ₜ₋₁) / 當日最大投入金額ₜ
```

【補】原文「淨利, - 淨利,,」已修正為「淨利ₜ − 淨利ₜ₋₁」。

**時間加權 RNₜ%**：

```
時間加權RN%ₜ = (淨利ₜ − 淨利ₜ₋₁) / 當日再投資金額ₜ
```

**當日再投資金額ₜ**：
= 重新買入策略內所有部位（含未平倉、當日平倉）所需金額

```
當日再投資金額ₜ = t日開倉部位的開倉金額 + t-1日前開倉的部位重新買入金額

t日開倉部位          = 開倉金額ₜ
t-1日前開倉的部位重新買入金額 = ROUNDDOWN(收盤價ₜ₋₁ × ABS(交易數量) × Lots × IFF(固定數量, 槓桿倍數, 1), 0)
```

## 4. 主要計算公式

### 4.1 淨利 NetProfit

```
淨利ₜ = 已實現損益ₜ + 未實現損益ₜ

已實現損益ₜ = 歷史已實現獲利ₜ₋₁ + 歷史已實現虧損ₜ₋₁
            + 今日已實現獲利ₜ₋₁ + 今日已實現虧損ₜ₋₁
            + 今日已實現獲利ₜ + 今日已實現虧損ₜ

未實現損益ₜ = 未實現獲利ₜ + 未實現虧損ₜ
```

### 4.2 時間加權報酬率 TimeWeightedRateOfReturn

```
TWR%ₜ = (1 + TWR%ₜ₋₁) × (1 + 時間加權RN%ₜ) − 1
```

【補】原文「TWR%, = (1 + TWR%, 1) * (1 + 時間加權RN%, ) - 1」已修正為標準下標格式。

### 4.3 最大投入報酬率 MaxInvestRateOfReturn

```
MIR%ₜ = 淨利ₜ / 最大投入金額ₜ
```

### 4.4 買進持有報酬 BuyAndHoldRateOfReturn

```
買進持有報酬% = 所有回測商品依交易數量在回測區間起點買進、終點賣出的報酬率（含交易成本）
```

### 4.5 大盤指數報酬 TAIEXRateOfReturn

```
大盤指數報酬% = 加權指數(TSE.TW) 在回測區間起點買進、終點賣出的報酬率（含交易成本）
```

### 4.6 最大投入金額 MaxInvestAmount

```
最大投入金額ₜ = MAX(投入金額ₜ, 最大投入金額ₜ₋₁)

投入金額ₜ = Σ 未平倉開倉金額ₖ + 當日進場已平倉開倉金額ₜ − Σ 已實現損益ₖ
```

備註：以日為統計頻率，以持倉金額為運算基礎。

## 5. 損益統計

### 5.1 毛利 GrossProfit

```
毛利ₜ$ = 歷史已實現獲利ₜ₋₁ + 今日已實現獲利ₜ₋₁ + 今日已實現獲利ₜ + 未實現獲利ₜ
毛利ₜ% = 毛利ₜ$ / 最大投入金額ₜ
```

### 5.2 毛損 GrossLoss

```
毛損ₜ$ = 歷史已實現虧損ₜ₋₁ + 今日已實現虧損ₜ₋₁ + 今日已實現虧損ₜ + 未實現虧損ₜ
毛損ₜ% = 毛損ₜ$ / 最大投入金額ₜ
```

### 5.3 獲利因子 ProfitFactor

```
獲利因子ₜ = 毛利ₜ / ABS(毛損ₜ) × SIGN(淨利ₜ)
```

特殊情況：
- 僅獲利無虧損（分母為 0）→ `+∞`（回傳 +0）
- 僅虧損無獲利（分子為 0）→ `-∞`（回傳 -0）
- 無交易 → `n/a`

### 5.4 總交易成本 TotalTransactionCost

```
總交易成本ₜ$ = 歷史已實現交易成本ₜ₋₁ + 今日已實現交易成本ₜ₋₁ + 今日已實現交易成本ₜ + 未實現交易成本ₜ
總交易成本ₜ% = 總交易成本ₜ$ / 最大投入金額ₜ
```

## 6. 交易次數統計

### 6.1 總交易次數 TotalTradeCount

```
總交易次數ₜ = 已實現交易次數ₜ + 未實現交易次數ₜ
```

已實現包含：歷史及今日的獲利 / 虧損 / 損益二平交易次數。

### 6.2 獲利交易次數 NumberWinningTrades

```
獲利交易次數ₜ = 歷史已實現獲利交易次數ₜ₋₁ + 今日已實現獲利交易次數ₜ₋₁ + 今日已實現獲利交易次數ₜ + 未實現獲利交易次數ₜ
```

### 6.3 虧損交易次數 NumberLosingTrades

```
虧損交易次數ₜ = 歷史已實現虧損交易次數ₜ₋₁ + 今日已實現虧損交易次數ₜ₋₁ + 今日已實現虧損交易次數ₜ + 未實現虧損交易次數ₜ
```

### 6.4 勝率 PercentProfitable

```
勝率ₜ = 獲利交易次數ₜ / 總交易次數ₜ
```

## 7. 平均統計

### 7.1 平均交易 AvgTrade

```
平均交易ₜ$ = 淨利ₜ$ / 總交易次數ₜ
平均交易ₜ% = 報酬率ₜ% / 總交易次數ₜ
```

（時間加權與最大投入結果不同）

### 7.2 平均獲利交易 AvgWinTrade

```
平均獲利交易ₜ$ = 毛利ₜ$ / 獲利交易次數ₜ
平均獲利交易ₜ% = 毛利ₜ% / 獲利交易次數ₜ
```

### 7.3 平均虧損交易 AvgLossTrade

```
平均虧損交易ₜ$ = 毛損ₜ$ / 虧損交易次數ₜ
平均虧損交易ₜ% = 毛損ₜ% / 虧損交易次數ₜ
```

### 7.4 平均獲利虧損比 RatioAvgWinAvgLoss

```
平均獲利虧損比 = ABS(平均獲利交易 / 平均虧損交易)
```

## 8. 極值統計

### 8.1 最大獲利交易 LargestWinTrade

```
最大獲利交易$ = MAX(歷史已實現最大獲利, 今日已實現最大獲利, 未實現最大獲利)
最大獲利交易% = MAX(歷史已實現最大報酬, 今日已實現最大報酬, 未實現最大報酬)
```

### 8.2 最大虧損交易 LargestLosingTrade

```
最大虧損交易$ = MAX(歷史已實現最大虧損, 今日已實現最大虧損, 未實現最大虧損)
最大虧損交易% = MAX(歷史已實現最大報酬, 今日已實現最大報酬, 未實現最大報酬)
```

## 9. 連續統計

### 9.1 最大連續獲利 MaxConsecutiveProfit

```
當日淨利ₜ = 淨利ₜ − 淨利ₜ₋₁  【補：原文缺少減號】

連續獲利ₜ = IFF(當日淨利ₜ > 0, 當日淨利ₜ + 連續獲利ₜ₋₁, 0)

連續獲利%ₜ（最大投入算法）= 連續獲利$ₜ / 最大投入金額$ₜ
連續獲利%ₜ（時間加權算法）= IFF(時間加權RN%ₜ > 0, (1 + 時間加權RN%ₜ) × (1 + 連續獲利%ₜ₋₁) − 1, 0)

最大連續獲利%ₜ = MAX(連續獲利%ₜ, 最大連續獲利%ₜ₋₁)
最大連續獲利$ₜ = 最大連續獲利%ₜ 當日對應的金額
```

### 9.2 最大連續虧損 MaxConsecutiveLoss

```
當日淨利ₜ = 淨利ₜ − 淨利ₜ₋₁  【補：原文缺少減號】

連續虧損ₜ = IFF(當日淨利ₜ < 0, 當日淨利ₜ + 連續虧損ₜ₋₁, 0)

連續虧損%ₜ（最大投入算法）= 連續虧損$ₜ / 最大投入金額$ₜ
連續虧損%ₜ（時間加權算法）= IFF(時間加權RN%ₜ < 0, (1 + 時間加權RN%ₜ) × (1 + 連續虧損%ₜ₋₁) − 1, 0)

最大連續虧損%ₜ = MIN(連續虧損%ₜ, 最大連續虧損%ₜ₋₁)
最大連續虧損$ₜ = 最大連續虧損%ₜ 當日對應的金額
```

## 10. 區間統計

### 10.1 最大區間獲利 MaxRunUp

```
淨利低點ₜ = MIN(淨利ₜ, 淨利低點ₜ₋₁)
報酬率低點ₜ = MIN(報酬率ₜ, 報酬率低點ₜ₋₁)

衝高ₜ = IFF(淨利ₜ > 淨利低點ₜ, 淨利ₜ − 淨利低點ₜ, 0)  【補：原文缺少減號】

衝高%ₜ（最大投入算法）= 衝高$ₜ / 最大投入金額$ₜ
衝高%ₜ（時間加權算法）= IFF(時間加權報酬率ₜ > 報酬率低點ₜ, (1 + 時間加權報酬率ₜ) / (1 + 報酬率低點ₜ) − 1, 0)

最大區間獲利%ₜ = MAX(衝高%ₜ, 最大區間獲利%ₜ₋₁)
最大區間獲利$ₜ = 最大區間獲利%ₜ 當日對應的金額
```

### 10.2 最大區間虧損 MaxDrawdown

```
淨利高點ₜ = MAX(淨利ₜ, 淨利高點ₜ₋₁)
報酬率高點ₜ = MAX(報酬率ₜ, 報酬率高點ₜ₋₁)  【需確認：原文使用 MIN，疑為 MAX 之誤】

拉回ₜ = IFF(淨利ₜ < 淨利高點ₜ, 淨利ₜ − 淨利高點ₜ, 0)  【補：原文缺少減號】

拉回%ₜ（最大投入算法）= 拉回$ₜ / 最大投入金額$ₜ
拉回%ₜ（時間加權算法）= IFF(時間加權報酬率ₜ < 報酬率高點ₜ, (1 + 時間加權報酬率ₜ) / (1 + 報酬率高點ₜ) − 1, 0)

最大區間虧損%ₜ = MIN(拉回%ₜ, 最大區間虧損%ₜ₋₁)
最大區間虧損$ₜ = 最大區間虧損%ₜ 當日對應的金額
```

## 11. K 線與交易天數

### 11.1 回測 K 線根數 BarsInBackTesting

```
= 回測期間內的日 K 線總數
```

### 11.2 全部交易的平均持倉 K 線根數 AvgBarsInTrades

```
= SUM(每筆交易的持倉K線根數) / 總交易次數
```

### 11.3 獲利交易平均持倉 K 線根數 AvgBarsInWinningTrades

```
= SUM(每筆獲利交易的持倉K線根數) / 獲利交易次數
```

### 11.4 虧損交易平均持倉 K 線根數 AvgBarsInLosingTrades

```
= SUM(每筆虧損交易的持倉K線根數) / 虧損交易次數
```

### 11.5 實際發生交易天數 ActualTradingDays

```
= SUM(回測期間內部位 > 0 的天數)
```

### 11.6 交易天數佔回測區間比例 PercentageOfTradingDays

```
= 實際發生交易天數 / 回測K線根數
```

### 11.7 年均交易數 AnnualTradeCount

```
= ROUNDDOWN(總交易次數 / 年數, 0)
年數 = 完整年數 + 日曆日 / 365
```

**範例**：起始日 2023/01/01，今日 2024/05/22，總交易 100 次
→ 年數 = 1 + (31+29+31+30+22)/365 ≈ 1.39
→ 年均交易數 = ROUNDDOWN(100 / 1.39, 0) = 71 次

## 12. 報酬率統計

### 12.1 每筆報酬加總 SumOfIndividualReturns

```
= SUM(回測區間內每筆交易的報酬率)
單筆交易報酬率 = 獲利金額 / 開倉金額
```

### 12.2 平均單筆報酬 AvgOfIndividualReturns

```
= 每筆報酬加總 / 總交易次數
```

### 12.3 平均年化報酬 SimpleAnnualPercentageYield

```
平均年化報酬% = 報酬率 / (日期 − 策略起始日 + 1) × 365
平均年化報酬$ = 平均年化報酬% × 最大投入金額
```

（時間加權與最大投入結果不同）

### 12.4 複利年化報酬 GeometricAnnualPercentageYield

```
複利年化報酬$ = 複利年化報酬% × 最大投入金額
日頻率經過日期數 = 日期 − 策略起始日 + 1，總日期數 = 365
```

（時間加權與最大投入結果不同）

## 13. 風險指標

### 13.1 風報比 RiskReturnRatio

```
風報比 = 報酬率% / ABS(最大拉回%)
```

### 13.2 標準差 StandardDeviation

```
標準差 = SQRT(Σ(報酬率RN% − 報酬率RN%的平均值)² / (日數 − 1))
```

### 13.3 Beta

```
Beta = 相關係數(指數報酬率RN%, 報酬率RN%) × 報酬率RN%的標準差 / 指數報酬率RN%的標準差
```

### 13.4 Alpha（超額報酬）

```
Alpha = 報酬率RN%的平均值 × 總日期數
      − (一年無風險利率 + Beta × (指數RN%報酬率的平均值 × 總日期數 − 一年無風險利率))
```

### 13.5 夏普比率 SharpeRatio

```
Sharpe Ratio = (報酬率RN%的平均值 × 總日期數 − 一年無風險利率) / (標準差 × √總日期數)
總日期數 = 252
```

### 13.6 索提諾比率 SortinoRatio

```
Sortino Ratio = (報酬率RN%的平均值 × 總日期數 − 一年無風險利率) / (下行標準差 × √總日期數)

下行標準差 = SQRT(Σ(負報酬率RN% − 負報酬率RN%的平均值)² / (負報酬率RN%發生日數 − 1))
總日期數 = 252
```

## 14. 連續次數 ConsecutiveTimes

| Case | 條件 | 結果 |
|------|------|------|
| Case 1 | 已實現損益ₜ = 0 | 連續次數ₜ = 連續次數ₜ₋₁ |
| Case 2 | 已實現損益ₜ > 0 且 連續次數ₜ₋₁ ≠ 賺 | 連續次數ₜ = 賺1 |
| Case 3 | 已實現損益ₜ > 0 且 連續次數ₜ₋₁ = 賺 | 連續次數ₜ = 連續次數ₜ₋₁ + 1 |
| Case 4 | 已實現損益ₜ < 0 且 連續次數ₜ₋₁ ≠ 賠 | 連續次數ₜ = 賠1 |
| Case 5 | 已實現損益ₜ < 0 且 連續次數ₜ₋₁ = 賠 | 連續次數ₜ = 連續次數ₜ₋₁ + 1 |

### 最大連續獲利次數 MaximumConsecutiveGains

```
= MAX(所有賺錢的連續次數)
```

### 最大連續虧損次數 MaximumConsecutiveLosses

```
= MAX(所有賠錢的連續次數)
```

## 15. 創新高統計

### 15.1 創新高天數 DaysOfNewHighs

```
創新高天數ₜ = 創新高天數ₜ₋₁ + IFF(報酬率ₜ > 報酬率最大值ₜ₋₁, 1, 0)
報酬率最大值ₜ = MAX(報酬率ₜ, 報酬率最大值ₜ₋₁)，初始值 = 0%
```

### 15.2 未創新高天數 DaysWithoutNewHighs

```
未創新高天數ₜ = IFF(報酬率ₜ > 報酬率最大值ₜ₋₁, 0, 未創新高天數ₜ₋₁ + 1)
```

### 15.3 最大未創新高天數 MaximumDaysWithoutNewHighs

```
= MAX(每兩個相鄰創新高日之間的未創新高天數)
```

**計算範例**：回測 10 天，Day1、Day2、Day5、Day9 創新高：
- 起點～Day1 → 0 天
- Day1～Day2 → 0 天
- Day2～Day5 → 2 天
- Day5～Day9 → 3 天
- Day9～終點 → 1 天
- **最大未創新高天數 = MAX(0, 0, 2, 3, 1) = 3 天**

### 15.4 平均未創新高天數 AverageDaysWithoutNewHighs

```
= AVERAGE(每個區間的未創新高天數)
```

承上例：(0 + 0 + 2 + 3 + 1) / 5 = **1.2 天**

### 15.5 MDD 區間天數 MaxDrawdownDuration

```
MDD區間天數 = 最大區間虧損%發生日 − 最大區間虧損%累積區間起始日（往前推算最近一次回檔為 0 的日期）
```

## 16. 等比計算特殊規格

等比模式下，部分欄位的計算邏輯有所不同：

1. **不支援**：交易數量、金額（\$）、最大投入報酬率
2. 每筆交易的報酬率在每日所佔用的資金都**相同且獨立**
3. 同日多筆相同商品的報酬率**分開計算後加總**

### 16.1 等比模式的時間加權報酬率

```
TWR%ₜ = (1 + TWR%ₜ₋₁) × (1 + 時間加權RN%ₜ) − 1

時間加權RN%ₜ = SUM(每筆交易當日報酬率ₜ) / 總交易筆數 / 槓桿倍數
```

**每筆交易當日報酬率ₜ**：

| 情境 | 公式 |
|------|------|
| 進場日 | `出場還原價ₜ / 進場還原價ₜ − 買進成本% − 1` |
| 出場日 | `出場還原價ₜ / 出場還原價ₜ₋₁ × (1 − 賣出成本%) − 1` |
| 一般持倉日 | `出場還原價ₜ / 出場還原價ₜ₋₁ − 1` |
| 當沖交易 | `出場還原價ₜ / 進場還原價ₜ × (1 − 賣出成本%) − 買進成本% − 1` |

> 📌 **待確認**：「每筆交易當日報酬率（股息不再投入）」的計算方式仍待確認。

### 16.2 等比模式的股票交易成本%

```
買進成本% = 手續費%
賣出成本% = 手續費% + 交易稅%
```

### 16.3 等比模式的總交易成本

```
總交易成本 = SUM(每日交易成本)

每日交易成本ₜ = 權重ₜ × (買進成本% × 進場次數ₜ + 賣出成本% × 出場次數ₜ)
權重ₜ = 1 / 當日持倉筆數
```

**當日持倉筆數**：含當日新進場、當日平倉出場、跨日留倉（**當沖視為 1 筆，不重複計算**）

**計算範例：**

範例 1（一般進出場）：
- 新進場 2 筆 + 平倉出場 3 筆 + 留倉 5 筆 = 10 筆
- 權重 = 1/10
- 當日成本 = (1/10) × (買進成本% × 2 + 賣出成本% × 3)

範例 2（含當沖）：
- 新進場 1 筆 + 平倉 1 筆 + 當沖 2 筆 + 留倉 4 筆 = 8 筆
- 進場次數 = 1 + 2 = 3；出場次數 = 1 + 2 = 3
- 權重 = 1/8
- 當日成本 = (1/8) × (買進成本% × 3 + 賣出成本% × 3)

## 19. 週期分析計算規格

### 19.1 基本原則

- 僅支援「**最大投入報酬率**」算法
- 金額（\$）以每日報表的「獲利」、「總獲利」計算
- 單日損益 = 每日報表「獲利」欄位
- 區間損益 = 區間底總獲利 − 區間初總獲利

### 19.2 報酬率計算

```
單日報酬率 = 單日損益 / 當日帳戶金額 × 100%
區間報酬率 = Π(區間內的單日報酬率 + 1) − 1
```

### 19.3 主要指標公式

| 指標 | 金額（\$）| 百分比（%）|
|------|---------|-----------|
| 報酬率 / 淨利 | 淨利 / 總交易數 | 單期報酬率加總 / 總交易數 |
| 毛利 | 加總所選頻率下每期獲利 | 金額 / 最大投入金額 |
| 毛損 | 加總所選頻率下每期虧損 | 金額 / 最大投入金額 |
| 最大單期獲利 | MAX(單期損益)，全虧時回傳 0 | MAX(單期報酬率) |
| 最大單期虧損 | MIN(單期損益)，全賺時回傳 0 | MIN(單期報酬率) |
| 平均單期金額變動 | 淨利 / 總交易數 | `((Π(單期報酬率+1))^(1/總交易數) − 1) × 100%` |
| 平均單期獲利變動 | 毛利 / 獲利交易數 | `((Π(獲利期報酬率+1))^(1/獲利交易數) − 1) × 100%` |
| 平均單期虧損變動 | 毛損 / 虧損交易數 | `((Π(虧損期報酬率+1))^(1/虧損交易數) − 1) × 100%` |
| 平均獲利虧損比 | 平均獲利變動\$ / ABS(平均虧損變動\$) | — |
| 勝率 | 獲利交易數 / 總交易數 | — |
