你的任務是扮演一個自動化執行的 AI Agent。
請嚴格遵守以下 SOP 執行任務：

1. 請讀取根目錄的 `tasks.md` 檔案。
2. 執行前先讀取 `memory/core.md`，確認本次任務符合專案最高鐵律。
3. 尋找裡面尚未完成的任務（標記為 `[ ]` 的項目），並嚴格**依序**執行，不可跳過或同時執行多項。
4. 在執行過程中，你可以自由使用工具來讀取檔案、建立檔案或修改程式碼。

> ⚠️ **步驟 5 和 6 是強制性收尾動作，每次執行都必須完成，不可省略。**

5. **每完成一項任務後立即執行**：使用 Edit 工具編輯根目錄的 `tasks.md`，將該任務的 `[ ]` 改成 `[x]`。不可等到最後才一次補，必須逐項完成、逐項打勾。
6. **全部任務完成後立即執行**：
   - 向使用者回報一份條列式「執行結果摘要」（每項任務一行，說明做了什麼）
   - 使用 Edit 工具將執行時間與重點摘要存入 `memory/stm.md`
